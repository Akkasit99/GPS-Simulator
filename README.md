## การทำงาน (ภาพรวม)
- กำหนด “จุดเริ่ม” และ “ปลายทาง” (พิกัดหรือค้นหาสถานที่) แล้วคำนวณเส้นทางรถยนต์ผ่าน OSRM
- ระหว่างวิ่ง คำนวณตำแหน่งปัจจุบัน + ทิศทาง (Course) ตามความเร็วที่ตั้งไว้
- สร้างประโยค NMEA และส่งออกทาง Serial ตามช่วงเวลา (อิงค่า Hz/Log Time)
- เพิ่ม “จุดผ่าน” (Waypoint) ได้ด้วยคลิกขวาบนแผนที่ แล้วคำนวณเส้นทางใหม่ให้อัตโนมัติ

## คุณสมบัติ
- คำนวณเส้นทางจากจุดเริ่ม → ปลายทางอัตโนมัติ
- เพิ่มจุดผ่านด้วยคลิกขวา และคำนวณเส้นทางใหม่แบบลื่นไหล
- ปรับความเร็ว, ความถี่ส่งข้อมูล (Hz), และจำนวนดาวเทียมจำลอง
- เลือกโหมดเวลา: ใช้เวลาเครื่อง หรือกำหนดวัน/เวลาเอง
- ส่ง NMEA ออกทาง Serial: `GNGNS`, `GPRMC`, `GPGGA`, `GPSACP`

## สิ่งที่ต้องโหลด (Dependencies)

**ต้องมี**
- Python 3.9+ (แนะนำ 3.10+)
- Windows ที่มีพอร์ต COM (หรือ USB-to-Serial)

**ไลบรารี Python**
- customtkinter
- tkintermapview
- pillow
- requests
- pyserial

ติดตั้งแพ็กเกจ (ตัวอย่าง):
```bash
pip install customtkinter tkintermapview pillow requests pyserial
```

## วิธีรัน
โหมด GUI:
```bash
python gps_simulator.py
```
โหมด headless (รันจำลองแบบไม่มีหน้าต่าง):
```bash
python gps_simulator.py --headless --start "15.116999,102.205459" --end "15.083189,102.240478" --speed 60 --ticks 12
```
## การใช้งาน (GUI)
- กรอก “จุดเริ่ม” และ “ปลายทาง” เป็น `lat,lon` หรือพิมพ์ชื่อสถานที่แล้วกด “ค้นหา”
- คลิกขวาบนแผนที่เพื่อเพิ่ม/จัดการจุดผ่าน (Waypoint)
- ตั้งค่า Serial: เลือก COM และ Baud แล้วกด “เชื่อมต่อ”
- กด `START` เพื่อเริ่มวิ่งและเริ่มส่ง NMEA
- กด `STOP` เพื่อหยุด

## สิ่งที่แสดงออกมา (UI/Output)

**บนแผนที่**
- จุดเริ่มต้น / ปลายทาง / จุดผ่าน (Waypoint)
- ไอคอนรถวิ่งตามเส้นทาง

**บนแผงควบคุม**
- ความเร็วปัจจุบัน (ตัวเลขใหญ่) และ Altitude
- ระยะทางรวม, ข้อมูลช่วงความเร็ว, เวลาเดินทางโดยประมาณ, เวลา ETA

**Log/Serial**
- Log: แสดงประโยค NMEA ที่กำลังส่ง (จำกัดความถี่การแสดงผลเพื่อไม่ให้ UI หน่วง)
- Serial: ส่งเป็น ASCII และคั่นบรรทัดด้วย `\r\n`

## เวลา (Time Mode)
- โหมด “ใช้เวลาเครื่อง”: ใช้เวลา UTC ของเครื่องเป็นฐานเวลา
- โหมด “กำหนดเอง”: ระบุวันและเวลาเริ่มส่ง (วัน/เวลาแยกคนละช่อง) แล้วเวลาจะเดินต่อไปตามเวลาจริงหลังเริ่มวิ่ง

หมายเหตุ: เวลาที่ส่งออกใน NMEA เป็น UTC (ไม่ชดเชย UTC+7)

## ข้อมูลที่ส่งออกทาง Serial (NMEA)

เมื่อเริ่มวิ่ง โปรแกรมจะส่ง 4 ประโยคนี้ออกไปทุกช่วงเวลา ตามค่า Hz/Log Time:
- `$GNGNS,...*XX`
- `$GPRMC,...*XX`
- `$GPGGA,...*XX`
- `$GPSACP,...*XX`

โค้ดที่สร้างประโยคและส่ง Serial อยู่ที่ [gps_simulator.py](gps_simulator.py) ฟังก์ชัน `schedule_nmea()`

## ไฟล์ตั้งค่า

โปรแกรมจะอ่าน/เขียนค่าไว้ที่ `config.json` เช่น:
- start, end
- speed, hz, log_sec
- time_mode, base_date, base_clock
- sat_total
- serial_port, serial_baud
